# 2026-06-04 천년경영 탭 빈 화면 픽스 (st.stop 순서)

## 증상
천년경영 업로드 탭 선택 시 본문이 완전히 빈 화면(탭 제목만 보임). 파일 업로더·캡션 등 아무것도 안 뜸.

## 원인
`app/pages/1_파일처리.py`에서 탭 블록 순서가 basic → order(발주서) → cy(천년경영). 발주서 탭 블록은 상품관리 미확인/매출통계 미업로드 시 `st.stop()` 호출. st.stop()은 탭 하나가 아니라 **스크립트 전체 실행을 중단**하므로, 코드상 뒤에 있는 천년경영(cy) 탭 블록이 렌더되기 전에 중단됨. 첫 로드 시 상품관리 체크박스 미확인 → st.stop() 발동 → cy 미렌더.

## 수정
탭 블록 재배치: basic → cy(천년경영) → order(발주서). st.stop() 가진 블록을 맨 뒤로. 페이지 .py라 리부트 불필요(자동 반영). 구문 검증 후 배포.
