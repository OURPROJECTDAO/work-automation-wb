# 2026-06-24 식봄 행사기획 3차 — 행사가 인하여력 분석 + 식봄 GraphQL 경쟁가 크롤링 실증

## 무엇 / 왜
- 식봄측이 "7월 기획전 양식 시트의 행사가에서 단가 추가 인하 가능?" 문의 → 내부 검토 시트 제작.
- 대상 = 식봄 공식양식(2차) 판매자칸(H:O)에 우리가 이미 채운 13품목(컨츄리타임 레몬에이드 + 새우칩 6 + 메밀칩 4 + 꽈배기 2).
- 추가 요청 = foodspring(식봄 플랫폼) "명가꽈배기" 검색 최저가 크롤링 가능여부 판별 + 그 최저가 기준 마진.

## 처리
1. **상품no→관리코드 매핑**: 폼 J열(식봄 상품no) → listing_sikbom.csv(06-23 15:11·592행) 코드 조인 → product_master 단위별 매입가(PC낱개=매입단가[8]·박스=박스매입단가[9]·N=hapo=1 전건). master 상품명 교차검증 전건 일치.
2. **현 행사가 마진**(식봄 정산식 7%·3000×0.967·2700): 6.3~8.6%. 새우칩/메밀칩 대부분 6~7%대 = 식봄 기준마진(8%) 이하로 이미 공격적. 추가 인하여력(현재가−3%하한가)=품목당 100~700원(메밀칩 100=사실상 바닥).
3. **★ 식봄 GraphQL 크롤링 실증(신규 발견)**: foodspring.co.kr 지오펜스 없음(해외IP 200). 가격=SPA GraphQL(api.foodspring.co.kr/v2/graphql). 막힘=① introspection 비활성 ② persisted operationName POST=403. **해결**: 게스트 세션 쿠키(GET /로 GUEST_SESSION/ci_session 획득) + JS번들(pages/search/all 청크)에 박힌 raw 쿼리텍스트(SearchResultGoodsListPCPaginationQuery·9275자) 직접 POST → 200. price{salePrice·originalPrice·appliedCoupon.price·discountRate}·vendor·deliveryFee·stock 수신.
4. **명가꽈배기 9판매자 실가격**: 우리(태동유통) 참깨/흑당 5,000=식봄 최저가. 차순위 디에이치 5,480(참깨)·다봄 5,690(흑당). → "최저가 기준 마진"=현재 5,000=8.6%(우리가 최저라 동치). 5,470까지 인상해도 최저 유지(13.6%). 즉시할인 −20%는 전 판매자 동일(=마켓보로 플랫폼 부담)→정산 미반영, 마진은 판매가 기준 정합.
