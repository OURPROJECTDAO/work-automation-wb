# 2026-06-22 데일리 대시보드 가격 변동 알림 — 매입가만 표시

## 무엇
데일리 대시보드(0b) '가격 변동 알림' 섹션을 **매입가 전용**으로 변경. 기존엔 '구분' 멀티셀렉트로 판매가·매입가 둘 다(기본 둘 다 선택) 표시·필터 가능했음.

## 왜
사용자: 판매가는 볼 필요 없음 → 매입가만 보고 싶음. (판매가 변동은 가격변경 작업 결과라 이미 알고 있음 / 매입가 변동이 마진 침식 트리거로 더 중요)

## 변경 (app repo `app/pages/0b_데일리대시보드.py`, page-only)
1. `st.multiselect("구분", ["판매가","매입가"], default=...)` 토글 **제거** → `_chg = _chg[_chg["구분"] == "매입가"]` 고정 필터.
2. 빈 안내 문구 "선택한 구분에 해당하는 변동이 없습니다." → "매입가 변동이 없습니다."
3. 표 컬럼에서 군더더기 `구분`(전부 '매입가') **제거** → 관리코드·상품명·박스재고·방향·전일가·금일가·변동률(%)·변동일.

★ 탐지(core `stock_history.detect_price_changes`)는 무변경 — 여전히 매입가·판매가 둘 다 탐지하고 page에서 매입가만 필터. core 안 건드림(판매가 재노출 원하면 page 1줄 복귀로 충분).
