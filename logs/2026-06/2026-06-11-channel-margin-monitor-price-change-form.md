# 2026-06-11 channel-margin-monitor — 가격 일괄변경 양식 내보내기 + 권장가 올림

## 무엇
채널마진모니터에 **스마트스토어 가격 일괄변경 시스템** 추가. 표에서 상품 선택 → CSV 또는 '가격 일괄변경 양식(.xlsx)' 내보내기. 양식은 선택 상품을 기준마진 달성가로 맞춰 그대로 재업로드 가능.
동시에 **권장가를 100원 내림 → 올림(ceil)** 으로 변경(기준마진 이상 보장, (B) 사용자 결정).

## 왜
미달 상품 가격을 수작업 엑셀([1]마스터 조인본)로 바꾸던 걸 앱에서 처리. 내림 권장가는 기준마진에 살짝 못 미쳐(예 6900→권장 6900 잔존) 올림으로 교정.

## 변경
### core (channel_margin_monitor.py)
- `_floor100`→`_ceil100`, 권장가 = ceil-100 (기준마진 이상 보장). docstring/주석 갱신.
- `adjust_price(판매가,즉시할인,포인트,target_net)`: ★할인 우선. 인상=즉시할인 먼저 축소→모자라면 판매가↑ / 인하=즉시할인 확대→모자라면 판매가↓. 포인트 불변.
- `compute_new_prices(rows,recs,pids)`: 체크 상품번호 → {pid:(새판매가,새할인)} + skip(권장가 없는 미매칭/기준미설정). target=권장가.
- `build_bulk_price_xlsx(raw_xlsx,new_prices,cfg)`: 원본 양식(전체 컬럼)에서 헤더(data_start 이전) 보존 + 체크행만 F(판매가)·BF(즉시할인)·BG(단위'원') 수정, 미체크/빈행 삭제 → (bytes,남긴수,원본없음목록).
- `append_rows_to_raw(raw,src,pids,cfg)`: 신규추가 시 저장 원본에 신규 상품번호 행 값 추가.

### page (6_채널마진모니터.py)
- 표를 st.dataframe→**st.data_editor + 선택 체크박스 열**. 선택은 상품번호 기준 session_state 유지(필터 바뀌어도). 전체선택/해제 버튼(현재 필터 대상, nonce로 editor 리시드). editor key=filter_sig+nonce.
- 내보내기 2버튼: **CSV**(선택 있으면 선택, 없으면 현재필터) / **가격 일괄변경 양식 생성**(선택 대상, 원본 .xlsx에서 build). 양식 bytes는 session_state 저장 후 download_button(다운로드 rerun 함정 회피) + 변경 미리보기(현재가→새가·방향).
- **상품관리 갱신에 원본 .xlsx 저장 추가**: 전체교체=업로드본 그대로 `reference/listing_<key>.xlsx` 커밋 / 신규추가=append_rows_to_raw로 신규행 추가 커밋. `_gh_bytes`(바이너리 raw fetch)·`_commit_raw` 신설.
