# 2026-06-08 대시보드 — 이익 모드 (택배비 = ERP 00-12 라인)

## 무엇
대시보드 📊 탭에 지표 토글(매출/이익) + 이익 모드 추가.

## 왜
온라인 팀 이익(매출−수수료−택배비−매입가, 이익률=이익/매입가) 가시화 요청.

## 발견 (실데이터 검산, decisions/0008)
- 수수료는 판매금액에 이미 반영(스마트스토어 판매금액=판매가×(1−수수료)).
- 판매금액−판매이익=매입가(4105행 전수 일치). ERP 이익률 분모=매입가.
- **택배비 = 관리코드 00-12 "C타입 택배비-3000" 라인**: 수량=송장(택배)건수, 판매이익=−(수량×3000). 택배비가 ERP에 음수이익으로 이미 차감됨. → 이익 = Σ판매이익(00-12 포함). 추가 입력/재적재 0.
