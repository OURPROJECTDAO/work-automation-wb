# 2026-06-22 데일리 이상치 표 — 금액 컬럼 정수화

## 무엇/왜
이상치/전체 표 금액 컬럼(매출(net)·원가·택배·마진)이 실제 소수값(택배 송장배분·낱개 원가에서 소수 발생)이라 `localized` 포맷이 소수점까지 표시 → 사용자 불요. 정수 반올림으로 소수점 제거(콤마는 유지). 마진율·기준%·박스(송장배분)는 소수 의미 있어 보존.

## 변경 (app `0b_데일리대시보드.py`, page-only)
- 표 빌드에서 `_disp[c] = _disp[c].round(0).astype("Int64")` (c∈매출·원가·택배·마진). format은 그대로 localized(콤마). 커밋 24be59e.
