# 2026-06-15 intelligence-layer 두뇌① — 마진 침식 탐지·가격 제시

## 무엇 / 왜
지능 레이어 두뇌① (ADR 0018, §6①). 최근 매입가가 올라 **채널 기준마진(baseline) 아래로 떨어졌는데 아직 가격을 안 고친** 상품을 8채널 통합으로 탐지 + 기준 회복 권장가. 이력 엔진(1a 가격이력)의 첫 의사결정 산출물.

## ★ 스코프 정정 (사용자 확정 2026-06-15)
- **base 매익률(원가 마크업)은 오프라인용 → 폐기.** 사용자가 관리하는 건 온라인 채널 → **채널별 baseline_margin** 기준이 맞음.
- **기간 = 최근 3개월(기본).** 사용자는 보통 1달 내 99% 직접 발견·재설정 → 묵은 침식 불필요. 도구 역할 = "8채널에 흩어져 놓친 것 + 진짜 매입가 변동 확인."
- 정렬 = **미달폭(기준대비 부족분) 큰 순 기본**, 매입 상승폭 순 토글.
- listing 스냅샷 신선도 전제 수용(저장 listing 판매가로 계산 → 오래되면 갱신 필요).

## 신호 / 로직
- (최근 N개월 매입단가 인상) ∩ (현재 채널 마진 < 기준마진, 즉 cmm 탐지 < -0.01). **이미 재설정한 건 탐지 ≥ 0 → 자동 제외.**
- 마진율·기준마진율·탐지·권장가는 **기존 channel_margin_monitor.compute_listing 그대로 재사용**(정합성). 침식 도구는 거기에 "최근 매입인상" 원인·시점 필터를 씌움.
- 조인 키 = **관리코드**(price_changes에도 관리코드 존재 → listing '코드'와 직조인). 관리코드 충돌은 매입Δ% 최대 상품코드 대표 + 코드수 표시.
