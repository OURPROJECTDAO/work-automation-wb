# 2026-06-19 기준마진율 최적화(두뇌④) 설계 확정 — 인터뷰로 정의 도출

## 무엇
"상품별·채널별로 가장 좋은 마진율은?"을 사용자와 인터뷰(Q1~Q11 + 노브 4종)로 정의 확정. **설계만 — 미구현.** 정본 = workflows/margin-optimizer.md, 근거 = decisions/0026.

## 왜
- 레버는 채널마진모니터 **기준마진율 시트 딱 하나**. P×C ≈ 5,000(상품 ~600 × 채널 8). 사람이 다 못 만짐 → 데이터가 각 P×C의 최적 기준마진율을 잡아주는 두뇌④ 필요.
- 두뇌③ 탄력성(채널 가격 A/B)을 "서술"에서 "처방(권장 기준마진율)"으로 올리는 단계.

## 핵심 정의 (인터뷰 결론)
- **목표 = 순이익(원) 최대화.** 괜찮은 범위면 볼륨 우선. 하한 = 나들커뮤니케이션 마진(가장 공격적 B2B 마트 채널). 운용인력 = 현재 무시. 채널 독립(새 수요 — 교차잠식 모델링 불요, 사용자 확정).
- **베이스(상품 적정 마진) = 순이익 누적 ~85% 차지하는 proven 채널들의 순이익 가중평균 마진.** (검증샘플 15-04 ≈ 6.4%)
- **채널 행동 = 볼륨 ×(마진 vs 베이스) 4분면**: 고볼륨(proven)=베이스 정의·유지 / 저볼륨·고마진=베이스로 **절반스텝** 끌어내림 / 저볼륨·저마진=hold-low(안 올림, 팔리면 재평가).
- **탄력성 판단 = 시장가 대신 채널 내부 비교**: 더 싼 채널이 더 잘 팔리는 게 아니면 가격 제약 아님 → 안 내림. 마진 흔든 이력 있으면 데이터 최적, 없으면 시도→측정 루프(관찰을 실험으로 — Gate 2·3).
- **출력 = P×C 권장 기준마진율 → 기존 채널마진모니터 기준마진율 편집 data_editor에 prefill → 사람 확인 후 저장.** 라이브 자동반영 없음.

## 노브 4종 (확정)
1. 저볼륨·고마진 끌어내림 = **절반 스텝**(베이스까지 거리의 1/2씩, 반응 보고 또 절반/되돌림). 외삽·일괄 인하 금지.
2. 회전(②) 인하 = **심각도 비례(소진예측일) + −2%p/사이클 캡 + 재고 풀리면 자동복귀**. 바닥=역마진 금지+제조사 단가하한(margin_floor).
3. 게이트 = **전부 사람 저장 거침**(라이브 자동 없음) + 🟢수락/🟡검토/🔴필수(충돌·|Δ|>3%p·매출목표/시즌·신호없음) 플래그.
4. 관망 임계 = 비중 **<1%는 제외 아니라 저우선 실험큐**. 소형 채널도 가격 테스트 줌 → **2회 무반응 시 park(쉼)**, 영구 아님(분기/원가/시즌 변동 시 재시도). 신규 상품·채널은 항상 실험.

## 라벨 로스터 (확정)
- 채택(자동): ⑤ 마진×볼륨 4분면(베이스 대비)·⑥ 채널편차·② 회전(장기소진만↓, 단방향). 채택(사람): ⑦ 상품 매출목표·⑧ 시즌(선물세트 기존 라벨).
- 보류: ① 시장가(nadl) — ESM 쿠폰 왜곡(우리 역산 gross ≠ 고객 실결제가) → 일단 빼고 sell-through로 해석. 고객 실결제가↔nadl 정의되면 복귀. ④ 재구매·⑨ 미끼 — 채널×상품 고객 데이터 구조화 선결.
- 제외(라벨 아님, 대시보드 알림만): ③ 원가추세(일시적 — 두뇌①·데일리가 이미 함).

## 결합 메커니즘
- **(b) 베이스+가감**(제품 라벨 ①②⑦이 가드레일/자세, 채널 라벨 ⑤⑥이 ±조정) + **(c) 진짜 충돌만 추천탭**. **(a) 단일 서열은 부적합** — 라벨이 서로 다른 그레인(제품 단위 vs 채널 단위)·다른 측정이라 한 줄로 못 세움(실증).

## 검증 (실데이터 인터뷰 중)
- 매출자료(정산 진실)+ship_alloc 실측 택배로 채널별 순마진·순이익 산출 = 정확(채널 단위). 비-EA 채널(나들)은 채널 총택배 정확·품목 분배만 물류량 추정.
- 31-04-02(코카콜라500 스마트): 18개월 마진 평탄(5~6%)·corr≈0 → 신호 없음 = 실험 대상(게이트가 정확히 걸러냄).
- 15-04(스팸클래식340·nadl 매칭): 순이익 96%가 상위 3채널(식봄·캐시·스마트), 베이스 ≈6.4%. 액션=캐시노트 마진↑(시장+9.4% 비싼데 물량 869=가격둔감·⑥) 1칸만 뜸 → "시스템이 건드릴 소수만 고른다" 실증.
- 나들커뮤니케이션: 마트 전용 B2B 채널·00-12=C타입 택배비 3000/송장·12개월 4.64억 → 총마진 10.6%→순마진 4.0%(택배가 매출의 6.7%). 박스내품 작은 품목(티오피 6팩 등) 직격.
