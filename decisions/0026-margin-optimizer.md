# ADR 0026 — 기준마진율 최적화(두뇌④): 정의·메커니즘 확정

날짜: 2026-06-19 · 상태: 설계확정·미구현 · 정본 detail: workflows/margin-optimizer.md

## 맥락
"상품별·채널별로 가장 좋은 마진율은?"을 정의해야 함. 우리가 쥔 레버는 채널마진모니터 **기준마진율 시트 하나**, 대상은 P×C ≈ 5,000(상품 ~600 × 채널 8) — 사람이 다 못 정함. 두뇌③(채널 가격 A/B·탄력성)은 "서술"까지만 와 있어, 이를 "권장 기준마진율(처방)"으로 올리는 결정이 필요. 사용자와 인터뷰(Q1~Q11 + 노브 4종)로 도출.

## 결정
1. **목표 = 순이익(원) 최대화**, 괜찮은 범위면 볼륨 우선. 하한 = 나들커뮤니케이션 마진(가장 공격적 B2B 마트). 운용인력 무시(현재). **채널 독립(새 수요) → 채널별 따로 최적화**(교차잠식 모델링 불요).
2. **베이스(상품 적정 마진) = 순이익 누적 ~85% proven 채널의 순이익 가중평균 마진.**
3. **채널 행동 = 볼륨 ×(마진 vs 베이스)**: 고볼륨=베이스 정의·유지 / 저볼륨·고마진=베이스로 절반스텝 인하 / 저볼륨·저마진=hold-low(안 올림).
4. **탄력성 판단 = 채널 내부 비교**(시장가 대신): 더 싼 채널이 더 안 팔리면 가격 제약 아님. 흔든 이력 없으면 시도→측정 실험 루프(Gate 2·3).
5. **노브**: ① 저볼륨·고마진=절반스텝 ② 회전=심각도비례·−2%p캡·자동복귀·역마진/제조사하한 클램프 ③ 게이트=전부 사람 저장+🟢🟡🔴(라이브 자동 없음) ④ 관망=비중<1% 저우선 실험큐(제외 아님)·2회 무반응 park(영구 아님)·신규 항상 실험.
6. **라벨**: 채택(자동 ⑤4분면·⑥채널편차·②회전 / 사람 ⑦매출목표·⑧시즌). 보류(①시장가=ESM쿠폰왜곡 / ④재구매·⑨미끼=고객데이터 선결). 제외(③원가추세→대시보드).
7. **결합 = (b) 베이스+가감 + (c) 충돌만 추천탭.** (a) 단일 서열 부적합(라벨 그레인 상이).
8. **출력 = P×C 권장 기준마진율 → 기존 cmm 기준마진율 편집창 prefill → 사람 저장 → 반응 측정 루프.**

## 근거
- 마진 정의를 기준마진율 시트와 동일(순이익/정산액·실택배비)하게 둬야 prefill 값이 바로 쓰임. ship_alloc 실측 택배 차감 필수(gross 마진은 과대).
- 베이스를 "잘 팔리는 채널 평균"으로: 이미 검증된 가격대가 그 상품의 적정선. 순이익 가중 = 목표(순이익)와 일관.
- 순이익 꼭짓점이 인하 한계(ε×마진=1). 단 꼭짓점 위치는 상품마다 달라 데이터로만 판정 → 관찰 데이터의 인과 한계를 시도→측정 루프로 보완.
- 관망을 "테스트 무반응" 상태로 정의(작다≠관망) → 콜드스타트 채널 영구배제 방지(사용자 지적).
- 시장가(①) 보류: ESM 등 쿠폰 채널은 역산 gross ≠ 실결제가 → 왜곡. sell-through 해석으로 대체.

## 대안 (기각)
- 분모=매입가(두뇌③ A/B 정의): 채널 비교엔 정직하나 기준마진율 시트(정산액 분모)와 불일치 → prefill 부적합.
- 절대 볼륨/개수 임계로 proven 판정: 상품마다 단가·물량 천차만별 → 상대(순이익 누적%)로.
- (a) 라벨 단일 서열: 제품 단위(①②⑦)와 채널 단위(⑤⑥)가 섞여 성립 안 함.
- 라이브 자동반영: Gate 1(사람은 결정만)에 반함 → 전부 사람 저장 거침.
- 한 방 인하/외삽: 탄력성 미지 채널에서 마진만 날림 → 절반스텝·실험.

## 검증 (실데이터)
- 15-04(스팸클래식340·nadl 매칭): 베이스≈6.4%, 순이익 96% 상위3채널, 액션=캐시노트↑ 1칸·ESM hold-low.
- 31-04-02(코카콜라500 스마트): 마진 평탄·corr≈0 → 실험 대상(게이트 정상).
- 나들커뮤니케이션: 택배 4.64억 → 순마진 4.0%(하한 채널 실체).
