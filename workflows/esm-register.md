# esm-register (ESM = G마켓/옥션 통합) — 플랫폼 인스턴스

> 공통 개념(낱개/박스·정제상품명·합포N·판매가공식·카테고리·이미지·서식일치)은
> **workflows/product-registration-common.md** 참조. 이 문서는 **ESM 전용값**만.

## ESM 전용 값
- **수수료율 17%**. 마진율 디폴트 10%(배치 입력). 배송 경제 공통(고객 3000/실비 2500).
- 판매가 = ⌊(매입가 + 2500) / [(1−0.17)(1−0.10)] − 3000⌋ 100원 내림. **분모 0.747**. 박스모드 매입가=박스매입단가.
- **G마켓 단독 등록**(옥션은 신규 업로드 안 함). 골든도 G마켓 전용.

## 양식 (NEW 일반상품)
- ESM 일괄등록 양식. **261열**, 시트명 `NEW 일반상품`. **데이터는 8행부터** 입력, **1~7행 삭제/변경 금지(오류)**. 예시행은 지우고 우리 것만.
- 행4=필드명 · 행5=필수여부 · 행6=입력안내 · 행7=ESM기본예시. A열=행번호(헤더 없음, 1..N).
- 채우는 **15열**(G마켓 전용):

| 열 | 항목 | 값 | 타입 |
|---|---|---|---|
| A | 행번호 | 1..N 순번 | int |
| B | 노출 사이트 | `G마켓` 고정 | str |
| D | G ID | `td680708` 고정 | str |
| E | 상품명 | 정제 박스명(스마트스토어 동일) | str (상품명+프로모션 합 ≤100byte, 한글2byte) |
| K | 카테고리 코드 | 카테고리표 ESM카테고리(20자리) | **str/`@`텍스트**(앞자리0 보존) |
| M | G 노출코드 | 카테고리표 사이트카테고리 | **str/`@`텍스트** |
| N | 판매기간 | `무제한` 고정 | str |
| P | G 판매가 | 공식 | int |
| Z | 기본이미지 | A1 URL | str |
| AB | 상품상세설명 | `<img src='…_B1.jpg' />` | str |
| AC | 배송정보 템플릿코드 | `15285`(유료) | int |
| AL | 상품군 코드 | `40`(가공식품) 고정 | int |
| AM | 상품고시정보 템플릿코드 | `146068` 고정 | int |
| BA | 원산지 상품타입 | `상세설명표기` 고정 | str |
| BB | 원산지 지역타입 | `기타` 고정 | str |

- 고정값: B=G마켓 · D=td680708 · N=무제한 · AC=15285 · AL=40 · AM=146068 · BA=상세설명표기 · BB=기타.
  - AL=40·AM=146068 은 **가공식품 기준**. 생활용품은 다르나 ESM엔 생활용품 업로드 안 하므로 항상 40/146068.
  - BK(부가세) 미입력 → 과세 자동(면세 아님).

## 카테고리 (K·M)
- 룩업표: **ESM_카테고리목록_식품**(진짜 OLE2 .xls, cp949, xlrd로 읽음). 291행. 컬럼: ESM카테고리명·사이트카테고리명·사이트(G마켓)·**ESM카테고리(=K)**·**사이트카테고리(=M, G노출코드)**. → K·M이 **한 행에 같이** 있음(노출코드 별도표 불필요).
- 글로벌하베스트 매핑(스마트스토어 분류와 일관):
  - 스낵 `과자/간식>스낵` K=00030002002300020000 / M=100000036200000370300003823 (새우칩·메밀칩·누룽지과자)
  - 녹차음료 `음료>차음료/전통음료>녹차음료` K=00030002000300100001 (그린티)
  - 기타차음료 `…>기타차음료/전통음료` K=00030002000300100016 (우롱티·자스민티)
  - 기타탄산음료 `음료>탄산/청량음료>기타탄산음료` K=00030002000300030006 (요구르트 스파클링 — 탄산)
  - 커피믹스 `커피/차류>커피/코코아>커피믹스` K=00030002000400010002 (동서 커피믹스)

## 배송정보 템플릿코드 (AC)
- **유료배송=15285 · 무료배송=15287**. 우리 상품은 유료배송(고객 3000원 수취) → **15285**.
- 합포장(N) 세부 배송설정은 **업로드 이후 사용자가 직접** → 업로드 단계에선 N→코드 매핑 불필요.

## 이미지
- 공통 호스트. 기본(Z) `https://gi.esmplus.com/td680708/{관리코드}/{관리코드}_A1.jpg`, 상세(AB) `<img src='…_B1.jpg' />`.
- 확장자 실검사: 글로벌하베스트 15품목 A1·B1 전부 `.jpg` 실재(HTTP 200) 확인. gi.esmplus.com bash 네트워크 접근 가능.

## 상품명 (E)
- 스마트스토어 정제 박스명 그대로. 박스 suffix `{박스내품}개`. [관리코드] 프리픽스 없음(이지어드민과 다름).
- ESM 제한: 상품명+프로모션 합 ≤100byte(한글 2byte). 현 배치 최대 39byte.
