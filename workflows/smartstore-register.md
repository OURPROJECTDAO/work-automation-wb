## 스마트스토어 전용 값
- **수수료율**: 6%.
- **배송 경제**: 공통(고객 3000 / 실비 2500).

## 양식(일괄등록) — 채우는 13개 컬럼 + 셀 타입/서식
> 양식 93열. 아래 13개만 채우고 나머지 빈칸 유지. 셀 타입/서식 100% 일치(공통 원칙).

| 컬럼 | 항목 | 값 | 타입/서식 |
|---|---|---|---|
| A | 판매자상품코드 | PC+상품코드 / 관리코드 | str / General |
| B | 카테고리코드 | 매핑 | **str**(숫자처럼 보여도) / General |
| C | 상품명 | 정제 | str / General |
| D | 상품상태 | 신상품(고정) | str |
| E | 판매가 | 공식 | **int** |
| F | 단위가격 사용여부 | N(고정) | str |
| J | 부가세 | 과세/면세(사용자 입력) | str |
| L | 재고수량 | 9999(고정) | int |
| W | 대표이미지 | URL | str |
| Y | 상세설명 | img 태그 | str |
| AD | 원산지코드 | 03(고정) | str + 서식 `@`(앞자리0 보존) |
| AI | 배송비템플릿코드 | N별 매핑 | **int** |
| BD | A/S템플릿코드 | 3379093(고정) | int |

## 고정값 (스마트스토어 양식)
- 상품상태=신상품 · 단위가격여부=N · 부가세=과세(면세는 사용자 지정) · 재고=9999 · 원산지=03(@텍스트) · A/S=3379093.
