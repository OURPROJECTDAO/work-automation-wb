# slow-moving-inventory (재고 회전 둔화상품)

> 챗 네이티브·앱 미탑재(사용자 확정 2026-06-24). 월 2회 정도 수동 실행. 산출=엑셀 3시트(3/6/12개월).

## 요약
돈 묶여 있는데 안 팔리고 재입고도 끊긴 재고를 잡아내는 리포트. 매 실행마다 골든 워크북에
`YYYYMMDD(N개월)` 시트(N=3·6·12)를 append. 각 시트는 그 N개월 평균 기준으로 독립 판정.

## 대상 필터 (3개 AND, 시트별 독립)
- 재고금액 ≥ 5,000,000원
- 입고경과일 ≥ 90일 (최근 실입고 후 3개월+ 미입고)
- 소진개월수(해당 N) ≥ 6개월
- 식품음료구분 ∈ {식품, 음료} — **선물세트·비식품(미분류) 제외** (골든 전 시트에 선물세트 0건으로 확인)
- 정렬 = 재고금액 내림차순. 택배비(상품명) 등 의사코드 제외.

## 컬럼 (8개·골든 헤더 그대로)
상품명 · 식품 음료 구분 · 잔여 재고 · 재고금액 · 소진개월수 · 입고경과일 · 판매량증감 · 월평균판매량(N개월내 평균치)

| 컬럼 | 산출 | 출처 |
|---|---|---|
| 잔여 재고 | 박스재고(박스) | product_master 박스 (상품관리 현재값) |
| 재고금액 | 박스재고 × 박스매입단가 | product_master (stockout.py와 동일식) |
| 월평균판매량(N) | 최근 N개월 매출 낱개합 ÷ 박스내품 ÷ N (박스/월) | 매출자료(거래일자·관리코드·수량) |
| 소진개월수 | 박스재고 ÷ 월평균판매량(N) | — |
| 입고경과일 | 오늘 − 최근 실입고일 | 매입현황(합계액>0 & 수량>0) |
| 판매량증감 | 최근 30일 낱개 ÷ 직전 30일 낱개 − 1 (N 무관, 직전 0이면 공란) | 매출자료 |
| 식품 음료 구분 | 식품/음료 | product_attributes.csv 식품음료 |

- ★ 단위 = **박스**(잔여·월평균·소진 전부 박스). 검증: 펭귄 682박스×20,400=13,912,800(재고금액)·682÷9.17≈74(월평균).
- 매출 anchor = 매출자료 최신 거래일(`max(거래일자)`). N개월 창 = (anchor − N월, anchor].
  "3/6/12 = 매출자료 최신성 기준" — 최신 매출일에서 N개월 역산.
- 매출 전무(월평균=0 → 소진 무한) 항목은 제외(골든은 항상 유한값). 월평균 tiny>0 → 큰 소진개월수로 표시(정상).
- 입고경과일·재고·재고금액 = 현재값(상품관리·매입자료 최신). 매출 창만 anchor 기준 — 사용자 확정.

## 데이터 소스 (전부 적립 완료·관리코드 직조인)
- 매출: work-automation-data master/sales_YYYY-MM.parquet (정산 진실원천·낱개)
- 매입: purchases/buyin_YYYY-MM.parquet (최근 실입고일)
- 재고/원가: app reference/product_master.csv (박스·박스매입단가·박스내품)
- 식품음료: app reference/product_attributes.csv

## 골든 서식 (다음부터 반드시 일치 — reference/slow_moving_inventory_golden.xlsx)
- 시트 3개만: `YYYYMMDD(3개월)`·`(6개월)`·`(12개월)`. **산출기준 등 부가 시트 금지.**
- 폰트 = 맑은 고딕. 헤더 14 bold, 데이터 11.
- 헤더: 채움 없음(흰 배경)·**thin 테두리(헤더 행만)**·col1·col2 좌측/세로중앙·col3~8 가운데. **col8(월평균) 헤더만 빨강 글자(FFFF0000)**, 나머지 검정.
- 데이터: 테두리 없음·채움 없음. col1(상품명) 좌측, col2~8 가운데.
- 숫자서식: 잔여재고=General(정수) · 재고금액=`"₩"#,##0;[Red]\-"₩"#,##0` · 소진개월수=`0.0` · 입고경과일=General(정수) · 판매량증감=`0%`(값은 비율 0.9→90%) · 월평균=General.
- 열너비 ≈ A 45·B 18.6·C 12.4·D 11.5·E 14·H 37 (F·G 기본). 틀고정 A2.

## 운영 / 입력
- 입력 = 상품관리(product_master)·매입자료(매입현황)·매출자료. 사용자가 최신화 후 실행.
- 6월처럼 부분월 매출(영업이익현황 export)을 1회성으로 더할 땐 **parquet 미적재(메모리 합산만)** — 부분월·00-12 누락은 슬로우무빙에 무해(택배비 코드 제외라). 정식 풀버전은 월적재로 따로.
- 영업이익현황 파싱 = core/dashboard/sales_data.py parse_sales 동형(합계행=거래일자 NaT 제외·KEEP_COLS).

## 검증 / 함정
- 잔여재고·월평균·소진 단위는 전부 박스 — 낱개로 착각 금지.
- 선물세트(product_attributes 식품음료=선물세트 65종) 반드시 제외.
- 입고경과일 90·소진 6 경계 = 골든 최저값(입고 91~·소진 6.0~)에서 확정.
- 1회 실행 = 골든 워크북에 시트 3개 append(과거 시트 보존).
